PACKMAN GAME
=========================

# To run the game :

- First install OpenGl on the os which you are using.

## For mac
  ```
  mkdir build
  cd build
  cmake ..
  make
  ./graphics_asgn1
  ```

## For Linux
- Rename the cmake_for_linux.txt to CMakeLists.txt .
- Apply same procedure as above for mac then.
